_Finding AssignmExit_ is a 3d computer game that offers both FPS and TPS modes. In this game, the game player must escape a complex and mysterious maze to submit assignments. Go through the maze and reach the wonderful ending. It was created using open source software such as Unreal Engine and Blender. The maze was created using the Binary Tree algorithm with Python code and visualized with Maya, a 3d modeling program.
